# DGS2B
### DG-LAB-E-STIM-Power-Box-V3-SOCKET-Server-To-V2-Bluetooth-Converter
---
## 介绍：

这是一个非常简单的小Python脚本，用于让2.0主机支持socket控制

它目前还有许多问题，比如当前强度数据没有回传到服务器等等，总之，欢迎提出pr

未来可能会添加图形用户界面和指令控制
